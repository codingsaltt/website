# Personal Profile Website

A clean and modern personal profile website built with React.

## About

This is the personal profile site for **@codingsalt** - Web Developer & Content Creator.

## Features

- Modern, minimalist design
- Responsive profile card
- Custom SVG avatar
- Smooth hover animations
- Clean gradient background
- PWA ready with custom icons

## Tech Stack

- React.js
- CSS3 (Custom styling)
- SVG graphics

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000)

### Build

```bash
npm run build
```

Builds the app for production to the `build` folder.
